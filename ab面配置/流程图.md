# A/B 面配置流程图

> 现行金样：`tools/browser/pdf` · 对照：`videodownload v1.2.0`

## 冷启双轨（PDF：Application 两协程）

```mermaid
flowchart TD
    App["Application.onCreate"] --> CorA["协程A: AppAdsBootstrap.run"]
    App --> CorB["协程B: assetsA + MonetizationKit.init"]
    CorA --> ST["startSettlement"]
    ST --> FC["FC轨 8s×最多3次"]
    ST --> A1["归因阶段1 6s"]
    FC --> fcR["fcReady"]
    A1 --> onR["onReady"]
    onR --> Wait["awaitFc 30s"]
    fcR --> Wait
    Wait --> M2["Mode2判面"]
    M2 --> C["commitAbFace"]
    C --> BR{"B面远程生效?"}
    BR -->|否| Retry["B面补拉最多3次"]
    BR -->|是| A2
    Retry --> A2{"阶段2?"}
    A2 -->|是| P2["阶梯30s/2m/5m/14m"]
    P2 -->|拉到买量| MID["applyModeUpdateAfterExtended\n中途升B"]
    MID --> OK2["JSON+补货+listener"]
    A2 -->|否| End["结束"]
```

## 中途升 B（必接）

```mermaid
flowchart TD
    T["触发: 阶段2 refer / commit B / FC总开关=1"] --> M["isModeB false→true"]
    M --> RC["applyRemoteConfigCore\nB 广告 JSON"]
    RC --> FCR["schedulePreloadAfterRemoteConfigRefresh"]
    RC --> NR["notifyAdRemoteConfigRefreshed"]
    M --> NU["notifyModeBUpgraded"]
    NU --> CO["schedulePreloadAfterLoadingOnBootstrapComplete"]
    NU --> PG["各页 listener"]
    FCR --> PL["runPreloadAfterLoading"]
    CO --> PL
    PG --> UI["preload + bind/show"]
    PL --> OK["B 专属位可展示"]
    UI --> OK
```

**禁止**：仅改 `isModeB`、无 Coordinator 补货、无页面 listener。
