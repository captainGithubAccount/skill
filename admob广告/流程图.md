# AdMob 广告流程

## 用户指定位置 → 落地

```mermaid
flowchart TD
    U["用户填广告位清单"] --> M["映射 AdSense + JSON"]
    M --> AB["标记 A/B 专属位"]
    AB --> P["写 preload 时机"]
    P --> S["写 bindNativeAd / showAd / obtainForShow(开屏) / Banner"]
    S --> V["checklist + Logcat 验收"]
```

## 插屏 / 原生

```mermaid
sequenceDiagram
    participant Page as 页面A
    participant Gate as canShowAd
    participant Loader as AdLoader
    participant PageB as 页面B
    Page->>Gate: enableFor + AB
    Gate-->>Page: true
    Page->>Loader: preloadAd(插屏/原生)
    PageB->>Loader: showAd / bindNativeAd
    Loader-->>PageB: 仅消费缓存
```

## 配置生效（PDF）

```mermaid
flowchart LR
    App["Application\napplyDefaultLocalAssetsA"] --> Mem["AdRemoteConfigManager"]
    AB["bootstrap commit\napplyByMode"] --> Mem
    FirebaseA["Firebase pdf_ad_config_a"] --> Mem
    FirebaseB["Firebase pdf_ad_config_b"] --> Mem
    Mem --> Enable["enableFor(sense)"]
    Enable --> Host["canShowAd\n+ AB 专属位"]
    Host --> Show["preload / show"]
```

## 双闸门 + 展示链

```mermaid
flowchart TD
    Init["MonetizationKit.isInit"] --> Gate1{isUmpResolved?}
    Gate1 -->|否| Skip["AdRequestLog 预加载跳过"]
    Gate1 -->|是| Gate2{allowsAdSense + ad_id?}
    Gate2 -->|否| VMod["VModifyLog 远程JSON未开放"]
    Gate2 -->|是| Gate3{B 专属位且 A 面?}
    Gate3 -->|是| Skip2["canShowAd false"]
    Gate3 -->|否| Req["发起 preload/load/show"]
```

## 开屏 Loading（PDF 金样）

```mermaid
sequenceDiagram
    participant S as SplashLaunchPipeline
    participant UMP as UMP
    participant C as AdPreloadCoordinator
    participant MK as MonetizationKit
    participant L as SplashAdLoader

    S->>UMP: awaitConsent
    S->>C: preloadAfterUmpConsent（无开屏）
    S->>MK: runWhenSdkInitializedOnce
    MK->>L: preloadAd 开屏 ×1
    S->>C: schedulePreloadBackground
    loop 放行闸 ≥2s 且 缓存或10s
        S->>L: isReady?
    end
    S->>L: obtainForShow
    alt 有缓存
        S->>S: show → 下一页
    else 无缓存
        S->>S: 直接下一页
    end
```

详见 [splash-loading.md](splash-loading.md)、[sdk-init-callback.md](sdk-init-callback.md)。
