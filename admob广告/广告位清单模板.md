# 广告位清单模板（给用户 / AI 填写）

接入或加广告时，**先填本表**（可复制到对话）。AI 据此生成 `AdSense`、JSON、页面代码。

## 填写说明

| 列 | 填什么 |
|----|--------|
| 页面 | Activity / Fragment 类名 |
| 广告类型 | 开屏 / 插屏 / 原生 / Banner |
| 展示时机 | 如：onCreate、按钮点击、离开页面前 |
| 预加载时机 | 如：上一页 onCreate、本页 onResume 前；Banner 填「现场 load」 |
| 容器 | 原生/Banner 填 `ViewGroup` id；插屏/开屏填 `-` |
| 大原生 | 原生是否 `useLargeLayout=true` |
| A/B | 仅 A / 仅 B / A+B |
| 频次/条件 | 如：enter 概率 50%、底栏每 N 次切换 |
| 备注 | 是否依赖订阅、FC 远程 key 等 |

## AI 收到清单后的动作

1. 为每行分配 `AdSense` id（PDF 已占 1–8，新项目从 9 起）
2. 更新 `assets/*.json` 与 Firebase `pdf_ad_config_*`（或项目自定义 key）
3. 仅 B 面位加入 `modeBExclusive`
4. 在对应页面插入 preload + show（Banner 写 Controller；用 templates）
5. 如需频次：用 `app_level_limit` 或 `AdNavigationCoordinator` / App 层计数
6. 输出「广告位对照表」给用户确认
