# A/B 面判断 — 流程图

> V3.1.0 定稿 · 树形简图  
> `isMode2()` 在 **Release / Debug 完全相同**；Debug 仅多一层 `DebugAbOverride` 方便测试强制 A/B。

---

## 1. 总览：启动 → isModeB

```
冷启动
  └── AppAdsBootstrap.run()
        ├── 拉取 Firebase Remote Config
        └── Mode2Utils.initMode2AndGet()
              │
              ├─ 缓存：上次已判 B 或 检测>4次 → 直接用 MMKV 缓存
              │
              └─ isMode2() → naturalModeB   （Release / Debug 相同）
                    │
                    ├── Release 包 → isModeB = naturalModeB
                    │
                    └── Debug 包  → DebugAbOverride.resolveModeB()（仅测试覆盖）
                          ├── FORCE_A  → isModeB = A
                          ├── FORCE_B  → isModeB = B
                          └── NORMAL   → isModeB = naturalModeB
```

---

## 2. isMode2() 判定（Release / Debug 共用）

```
启动 → Mode2Utils.initMode2AndGet()
  │
  ├─ 缓存：上次已判 B 或 检测>4次 → 直接用缓存
  │
  └─ isMode2() 判定
         │
         ├─【总开关 RC】enable_mode2_with_video
         │    ├─ RC 存在 且 =1  → 强制 B
         │    ├─ RC 存在 且 =0  → 强制 A
         │    └─ RC 不存在，或存在且值≠0且≠1 → 进入代码判断
         │
         ├─【代码判断】
         │    │
         │    ├─ 买量渠道检查
         │    │    子项 RC 不存在 → 视为开启，执行检查
         │    │    子项 RC 存在且=0 → 跳过
         │    │    子项 RC 存在且=1 → 执行检查
         │    │    isInstallFromAd() 未通过 → 记一次失败
         │    │
         │    ├─ GP 安装检查（规则同上）
         │    │    isInstalledFromGooglePlay() 未通过 → 记一次失败
         │    │
         │    ├─ 有执行过的检查 且 任一失败 → A
         │    └─ 有执行过的检查 且 全部通过 → B
         │
         └─【兜底】→ A  → naturalModeB
```

---

## 3. Debug 专属：DebugAbOverride 覆盖层

> 仅 **Debug 包**生效；**不改变** `isMode2()`，只覆盖最终 **`AppAdsBootstrap.isModeB`**。

### 3.1 resolveModeB（最终面）

```
naturalModeB（isMode2 自然结果，Release/Debug 相同）
  │
  └─ DebugAbOverride.resolveModeB(naturalModeB)
         │
         ├─ Release 包 → 直接 return naturalModeB
         │
         └─ Debug 包 → DebugAbOverride.get()
                │
                ├─ CODE_OVERRIDE_MODE == FORCE_A  → isModeB = false
                ├─ CODE_OVERRIDE_MODE == FORCE_B  → isModeB = true
                │
                └─ CODE_OVERRIDE_MODE == NORMAL
                       ├─ MMKV == FORCE_A  → isModeB = false
                       ├─ MMKV == FORCE_B  → isModeB = true
                       └─ MMKV == NORMAL   → isModeB = naturalModeB
```

### 3.2 get() 优先级

```
BuildConfig.DEBUG == false  → 恒 NORMAL

Debug 且 CODE_OVERRIDE_MODE != NORMAL  → 返回 CODE_OVERRIDE_MODE（弹窗/MMKV 无效）

否则  → MMKV KEY_DEBUG_AB_OVERRIDE（默认 NORMAL）
```

### 3.3 首页 AB 测试 UI 流程

```
冷启动 StartActivity
  └─ DebugAbOverride.onStartActivityCreate(intent)
         ├─ 桌面拉起（非 restart）→ 清除 KEY_DEBUG_AB_ENTRY_DISMISSED → 首页可再显示 AB 按钮
         └─ restartApp 拉起（EXTRA_DEBUG_AB_RESTART）→ 保持「已消费」→ 隐藏 AB 按钮

首页 HomeFragment
  └─ shouldShowHomeDebugAbButton()?
         ├─ CODE_OVERRIDE 锁定 A/B → 不显示按钮
         └─ 未 dismissed → 显示 AB 按钮 → DebugAbOverrideDialog
                ├─ 选 FORCE_A / FORCE_B / NORMAL
                ├─ set(mode)；若与当前不同 → restartApp()
                └─ markHomeDebugAbEntryConsumed() → 隐藏按钮

set(NORMAL) 时 → Mode2Utils.resetCachedCheckForDebug() → 下次冷启动重跑 isMode2()
```

优先级总览：

```
CODE_OVERRIDE_MODE（编译期）
    > MMKV KEY_DEBUG_AB_OVERRIDE（弹窗）
        > naturalModeB（Mode2Utils）
```

## 4. Release vs Debug 一张表

| 环节 | Release | Debug |
|------|---------|-------|
| `isMode2()` 自然判定 | 总开关 RC → 代码判断 → 兜底 A | **完全相同** |
| 最终 `isModeB` | = naturalModeB | 可被 DebugAbOverride 强制 A/B |
| 测试手段 | 无 | `CODE_OVERRIDE_MODE` / 首页 AB 弹窗 + `restartApp` |
| 读面字段 | `isModeB` = natural | `isModeB` = resolve 后（可与 natural 不同） |

## 5. 典型结果速查（naturalModeB，Release/Debug 相同）

| 总开关 | 买量 | GP | naturalModeB |
|--------|------|-----|--------------|
| 存在=1 | — | — | **B** |
| 存在=0 | — | — | **A** |
| 存在且≠0≠1 | ✅ | ✅ | **B**（代码判断） |
| 不存在 | ✅ | ❌ | **A** |
| 不存在 | ❌ | ✅ | **A** |
| 不存在 | ✅ | ✅ | **B** |
| 不存在 | ❌ | ❌ | **A** |

买量/GP：✅=该项已执行且通过，❌=已执行且未通过；子项 RC=0 跳过不计入。

### Debug 覆盖后 isModeB（与 naturalModeB 无关）

| 覆盖方式 | isModeB |
|----------|---------|
| NORMAL | = naturalModeB |
| `CODE_OVERRIDE_MODE` / MMKV **FORCE_A** | **A**（`false`） |
| `CODE_OVERRIDE_MODE` / MMKV **FORCE_B** | **B**（`true`） |

示例：naturalModeB=B 且 MMKV=FORCE_A → **isModeB=A**（用于测 A 面广告/通知）。
