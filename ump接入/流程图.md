# UMP 同意弹窗 — 流程图

> 定稿 · 先读缓存 → 无缓存再走 UMP  
> **可移植包**：`skill/ump接入/portable/` · 金样：`tools/browser/pdf`

---

## 1. 冷启动总览（PDF：SplashLaunchPipeline）

```
Application.onCreate（IO 并行）
  ├── MonetizationKit.prepareBeforeConsent()
  ├── MonetizationKit.init()  → isInit = true（与 UMP 并行）
  └── [Debug] AdConsentManager.debugConfig + AdTestDeviceIdLog

SplashActivity：2s 品牌动画 → SplashLaunchPipeline.start()
  ├── resetUmpResolvedForLaunchCycle()
  ├── [热启动] markUmpResolved()
  └── runPipeline()
        ├─ applyUmpProgressVisibility（willRunUmpGather → 藏进度条、显转圈）
        ├─ awaitConsent() → 见 §2
        ├─ preloadAfterLoading + loadSplashIfEnabled
        └─ awaitReleaseGate() → showSplashOrNavigate()
```

**videodownload** 另见 `StartActivity`：`ProgressDriver` + `runUmpWaitUiLoop` + `awaitSdkInitIfNeeded`。

## 2. AdConsentManager（先缓存）

```
requestGatherConsentAndInitAds { onUmpFlowFinished ->
  │
  ├─ ① hasCachedUmpConclusion?
  │     ├─ flow_completed_once == true
  │     └─ 或 SDK ∈ {OBTAINED, NOT_REQUIRED}
  │     └─ YES → markUmpResolved → 回调 → END（不弹窗）
  │
  ├─ ② !EuRegionHelper.shouldInitializeUmp?
  │     └─ YES → markUmpResolved → 回调 → END
  │
  └─ ③ gatherConsent
        ├─ [Debug] setDebugGeography(EEA) + addTestDeviceHashedId
        ├─ requestConsentInfoUpdate
        │     ├─ isConsentFormAvailable → ump_pop_show
        │     └─ loadAndShowConsentFormIfRequired（SDK 弹窗）
        ├─ markUmpConclusionCached
        ├─ markUmpResolved
        └─ onUmpFlowFinished → END
```

## 3. 弹窗展示决策（Debug 联调）

```
Debug 包 + 无缓存 + 进入 gather
  │
  ├─ AdMob Privacy & messaging 已 Publish?
  │     └─ NO → isConsentFormAvailable=false → 无 UI
  │
  ├─ setDebugGeography(EEA) 已设置?
  │     └─ NO → 可能 NOT_REQUIRED（尤其系统地区非欧盟）
  │
  ├─ 本机 hashed id 在测试列表?
  │     └─ 建议有，便于 Logcat 联调
  │
  └─ consentStatus=REQUIRED && formAvailable
        └─ loadAndShowConsentFormIfRequired → 弹窗
```

## 4. 双闸门（广告请求）

```
load / show / preload 前
  │
  ├─ MonetizationKit.isInit == true?        （Application）
  ├─ MonetizationKit.isUmpResolved == true? （UMP 结束）
  ├─ !isSubs
  └─ enableFor 远程配置等
        └─ 全部满足 → 发请求
```

## 5. UMP UI（PDF）

```
willRunUmpGather = !hasCachedUmpConclusion && 欧盟/英国

willRunUmpGather == true（冷启）
  ├─ addressbq4（横向进度）→ GONE
  └─ splashUmpWait（转圈，#3D7CFD）→ VISIBLE

UMP 结束
  ├─ addressbq4 → VISIBLE
  └─ splashUmpWait → GONE
```

---

## 6. 热启动

```
SplashLaunchPipeline.start / 等价入口
  ├─ resetUmpResolvedForLaunchCycle()
  ├─ isHotStart → markUmpResolved()（立即）
  ├─ 不 gather UMP
  └─ 不显示 UMP 转圈
```

---

## 7. 放行闸

```
tryPassReleaseGate
  ├─ consentResolved == false → 不放行（等 UMP）
  ├─ consentResolved == true
  │     ├─ elapsed >= 2s && adLoadFinished → 放行
  │     └─ UMP 后阶段超 10s → 丢弃开屏，仍放行（若已满 2s）
  └─ showSplash / navigate
```

---

## 8. 场景速查

| 场景 | gather | 转圈 | 弹窗 | 广告请求 |
|------|--------|------|------|----------|
| 有缓存（任意地区） | 否 | 否 | 否 | UMP 结束后允许 |
| 无缓存 + CN Release | 否 | 否 | 否 | 允许 |
| 无缓存 + Debug EEA 配置 + 后台 Publish | 是 | 是 | **可能** | gather 后允许 |
| 无缓存 + FR + 未 Publish | 是 | 是 | **否**（form 不可用） | gather 后允许 |
| 无缓存 + 拒绝 | 是（仅首次） | 是 | 首次可能 | **仍允许** |
| 热启动 | 否 | 否 | 否 | 立即允许 |
